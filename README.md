UPI Transactions Analysis Dashboar
Project Overview
This project focuses on analyzing UPI transaction data to understand digital payment trends and user behavior. The dashboard was created using Power BI to transform raw transaction data into meaningful insights through interactive visualizations.
The objective of this project is to explore how UPI transactions vary across different cities, banks, payment methods, and devices.

Dataset
The dataset used in this project contains information about UPI transactions such as:
1. Transaction ID
2. Transaction Date & Time
3. Transaction Amount
4. Sender Bank and Receiver Bank
5. City
6. Gender
7. Customer Age
8. Transaction Type
9. Payment Method
10. Device Type
11. Merchant Name
12. Transaction Status
13. Remaining Balance

Tools & Technologies
Power BI – Dashboard creation and data visualization
Microsoft Excel – Dataset storage and preprocessing

Dashboard Insights: 
The Power BI dashboard provides several key insights including:
Total number of transactions
Total transaction amount
City-wise transaction distribution
Gender-wise transaction analysis
Payment method usage
Device type distribution
Bank-wise transaction comparison
Transaction success vs failure analysis

Key Learnings
Data cleaning and preparation
Creating interactive dashboards in Power BI
Data visualization techniques
Understanding digital payment transaction patterns
